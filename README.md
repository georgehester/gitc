# Usage

## List

List the current .gitconfig's in the .gitc directory.

```console
foo@bar # gitc list
    Work
 -> Personal
```

## Switch

Switch which .gitconfig is currently being used.

```console
foo@bar # gitc switch
Select a git configuration
    Work
 -> Personal

Git configuration updated to Personal
```
